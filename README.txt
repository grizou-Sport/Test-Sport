Momentum v0.04 — Journal de mission sportive

Nouveautés :
- Suppression du menu latéral : navigation légère en haut uniquement.
- Hero narratif plein écran, avec image de fond personnalisable.
- La fenêtre du jour est fermée par défaut et s'ouvre uniquement au clic.
- Semaine vivante : 3 jours passés + aujourd'hui + 3 jours futurs.
- Bien-être intégré dans la page du jour.
- Correction de la création multiple lors de l'import GPX.
- Graphiques alimentés par les données de démonstration ou les données importées.

Image Hero :
Place ton fichier IMG_4510.JPG dans le même dossier que index.html.
Le CSS l'utilisera automatiquement comme fond du Hero.

Ouvrir :
Double-cliquer sur index.html.

Note :
Chart.js est encore chargé via CDN. Pour une version totalement locale, il faudra ajouter chart.umd.min.js dans le dossier et modifier la balise script.
