Momentum v0.01

Ouvrir index.html dans Safari, Chrome ou Edge.
Les séances ajoutées sont sauvegardées dans le navigateur via localStorage.
Utiliser Exporter JSON pour sauvegarder ou partager les données.
Utiliser Importer JSON pour reprendre une sauvegarde.

Pour modifier l'identité du projet sans coder : aller dans la section Objectif.
Pour modifier les sports : ouvrir app.js et compléter la liste sports au début du fichier.

Note : Chart.js est chargé via CDN pour cette version d'apprentissage.
Une prochaine version pourra intégrer chart.umd.min.js en local.
